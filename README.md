
# AI Agent Pencatatan Pengeluaran

## Prasyarat
- Node.js
- OpenAI API Key
- Twilio WhatsApp API
- Google Sheets API

## Instalasi
1. Clone repository
2. Install dependencies: `npm install`
3. Konfigurasi `.env`
4. Jalankan: `npm start`

## Fitur
- OCR Struk Otomatis
- Pencatatan Spreadsheet
- Insight Keuangan AI
